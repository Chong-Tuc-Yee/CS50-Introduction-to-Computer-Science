### Task: ###
**Implement a program to run simulations of the FIFA World Cup.**

### Specification: ###
Practice Question Source: [More Info](https://cs50.harvard.edu/x/2022/labs/6/)

### Usage: ###
To run program, user can execute following commands in codespace terminal:
1. `cd Pset6`
2. `cd world-cup`
3. `python tournament.py [csv file]`  (Eg. `python tournament.py 2018m.csv`)
